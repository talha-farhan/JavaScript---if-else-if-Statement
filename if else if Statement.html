JavaScript - if else if Statement

Things to Know

  • The else if statement specifies a new condition if the first condition is false
  • The ending is made using a simple else statement
  • The last else statement runs when no above condition is truthy

Syntax

if (condition1) { // block of code to be executed if condition1 is true } else if (condition2) { // block of code to be executed if the condition1 is false and condition2 is true } else { // block of code to be executed if the condition1 and condition2 are false }
